## 📌 About The Project

This project is a simple yet interactive **Quiz Application** developed in Python using the `tkinter` library.

It presents multiple-choice questions to the user, tracks answers, and displays the final score at the end.

This project is perfect for:
- 🐍 Python beginners
- 🎓 Students learning GUI development
- 💡 Practice for logic building

---

## ✨ Features

✅ Graphical User Interface (GUI)  
✅ Multiple Choice Questions  
✅ Real-time Answer Selection  
✅ Score Calculation  
✅ Result Popup Message  
✅ Beginner-Friendly Code
